# Photo to Beads

Project: Photo to Beads — a mobile Android application that converts photos into bead embroidery patterns.

Project Goal:
Create a convenient tool for crafters that allows users to:

convert a photo into a bead embroidery pattern

automatically select a bead palette

follow a step-by-step embroidery process

export patterns to PDF, PNG, or CSV

Target Audience:

bead crafters and embroidery enthusiasts

hobby craft communities

users of Perler / Hama / Miyuki / Preciosa beads

age: 25–60 years

Core Features (MVP)

Image Import

camera, gallery, PNG/JPG

crop, rotate, scale, select area

Pattern Generation

pipeline: image → resize → color quantization → palette mapping → pattern grid

settings: grid size (30×30, 50×50, 80×80, custom)

number of colors (5, 10, 20, 40)

bead type (round, square)

Bead Palettes

support for brands: Miyuki, Preciosa, Toho, Perler, Hama

each color includes: code, name, RGB, preview

Pattern Viewer

modes: color view, symbol view, embroidery mode

features: zoom, pan, highlight colors

Embroidery Mode (killer feature)

interactive pattern: mark stitched cells

progress tracking, highlight current color, filter by color

Materials Calculation

automatically calculates number of beads per color and total quantity

format: Red 12 – 350 beads, Blue 45 – 120 beads

Export

PDF, PNG, CSV

PDF contains: pattern image, color legend, materials list, grid

Project Potential

large audience of bead crafters

low competition in the craft app market

relevant SEO keywords: bead pattern, bead embroidery, bead design, bead from photo

long-term potential: iOS, web generator, patterns for diamond mosaics, cross-stitch, and pixel art
